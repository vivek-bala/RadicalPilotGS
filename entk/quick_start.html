RADICAL Pilot Quick Start Guide

Get Python Virtualenv.

Since RADICAL-Pilot is written in Python, Python Virtualenv is a requirement by RADICAL-Pilot. This how to install Python Virtualenv.

Get RADICAL-Pilot

Create a new VirtualEnv. Install using

The latest stable RADICAL-Pilot release is 0.40.3

$ virtualenv RADICALPilot
$ source RADICALPilot/bin/activate
(RADICALPilot)$ pip install radical.pilot

Run RADICAL-Pilot

Example RADICAL-Pilot scripts can be found in the RADICAL-Pilot repository example directory. Scripts have a .py file extension. Run the following commands to launch a simple hello world application.

$ wget https://github.com/radical-cybertools/radical.pilot/blob/devel/examples/00_getting_started.py
$ python 00_getting_started.py
