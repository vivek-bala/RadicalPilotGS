Ensemble Toolkit Quick Start Guide

Get Python Virtualenv.

Since Ensemble Toolkit is written in Python, Python Virtualenv is a requirement. This how to install Python Virtualenv.

Get Ensemble Toolkit

Create a new VirtualEnv. Install using 

	
			$ virtualenv myenv
			$ source myenv/bin/activate
			(myenv)$ pip install radical.ensemblemd

The latest stable Ensemble Toolkit release is 0.4.5

Run Ensemble Toolkit

We present here an example script that uses ensemble toolkit components to submit a Pipeline of Ensembles where each ensemble consists of tasks which perform a checksum on some given data. Run the following commands to download and launch the script.

	
		$ wget https://raw.githubusercontent.com/radical-cybertools/radical.ensemblemd/master/examples/pipeline_of_ensembles.py
		$ RADICAL_ENTK_VERBOSE=REPORT python pipeline_of_ensembles.py
